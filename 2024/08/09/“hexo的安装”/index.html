需要设置下面的内容
首先是确定要生成密钥的账户
ssh-keygen -t rsa -C “jikedao2024@163.com “
输入完以后弹出下面的内容
Generating public/private rsa key pair.
Enter file in which to save the key (/home/cloudide/.ssh/id_rsa):
一直敲回车即可，最后生成了一个秘钥
查看秘钥：cat ~/.ssh/id_rsa.pub
生成密钥成功以后，打开github，在右上角头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字任意，可按项目名称来，将第2步产生的密钥复制到框中，点击确定保存。
最后输入：ssh -T git@github.com ，如果没有报错，出现你的用户名，那就成功了。
